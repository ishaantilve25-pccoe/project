Objective

To analyze, compute, and classify the critical (stationary) points of real-valued functions of two independent variables $f(x, y)$ using the partial derivative test.

Expected Outcomes

  • Locate coordinate points where the function gradient vanishes ($\nabla f = 0$).
  • Utilize the discriminant criteria ($D = f_{xx}f_{yy} - f_{xy}^2$) to categorize stationary points.

Mathematical Context

For a continuous function $f(x,y)$, stationary points are determined by establishing where first-order partial derivatives equate to zero simultaneously:

f_x = ∂f/∂x = 0 and f_y = ∂f/∂y = 0

To classify these locations, evaluate the Second-Derivative Discriminant $D$ (Hessian Determinant):

D = r * t - s² Where: r = f_xx, s = f_xy, t = f_yy
  • If $D > 0$ and $r > 0$: Local Minimum.
  • If $D > 0$ and $r < 0$: Local Maximum.
  • If $D < 0$: Saddle Point.
  • If $D = 0$: The test is inconclusive.
